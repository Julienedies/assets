#前端开发环境简介
=====================================

##使用fis构建前端开发环境
详见：https://github.com/fex-team/fis

##环境搭建
1：在本机安装nodeJs;
2：执行fis-init.cmd初始化fis环境;

##使用
1：平常进行前端开发前，执行fis-dev.cmd命令;
2：上线部署前，执行fis-deploy.cmd命令;


##关于发布到线上
1：移除 fis --live 选项
1：修改静态资源域名为线上实际域名
